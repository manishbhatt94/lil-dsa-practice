
# dsa-practice

Maven Java 25 project containing sample LeetCode solutions with JUnit 5 tests.
Package root: `manish.dsa`

## How to run locally

Requirements:
- JDK 25 installed
- Maven installed (or import into IDE that understands Maven)

Build & run tests:

```
mvn test
```

Import into Eclipse / IntelliJ:
- Import as existing Maven project.
- Run tests from IDE or using `mvn test`.
